🔥 Flare Classification — When Can They Occur?

Solar flares are classified by peak X-ray flux. In this project, class is determined by the SoLEXS 6-10 keV ratio (peak flux / rolling background). Flares can occur at any time during observation.

⚡ How a Flare Happens — Anatomy

📡 1. Pre-Flare Enhancement

~15 min before peak, SoLEXS flux begins a gradual rise — a Gaussian bump centred ~900 s before peak. This is the precursor signal. SoLEXS channels rise by up to 13%.

Gaussian: e^{-(t+900)^2 / 2*450^2}

⚡ 2. Hard X-ray Burst

At the peak, HEL1OS detects a sharp impulsive spike. The burst is ~5× narrower than SXR. HXR typically leads the SXR peak — the Neupert effect.

HXR rise = 0.22 × SXR rise

📈 3. Soft X-ray Peak

SoLEXS flux rises as a Gaussian then decays exponentially. Peak ratio above background determines flare class (A/B/C/M/X).

Rise: Gaussian | Decay: e^{-t/tau}

📉 4. Decay & Return

Flux decays exponentially back to background. Decay times: 300 s (X-class) to 1100 s (C-class). Detection ends when ratio drops below 45% of threshold.

End: ratio < 0.45 × threshold

Continuously computes flux / rolling background on 6-10 keV channel.

Threshold Trigger

When ratio exceeds 3.0× AND flux is rising, a flare is flagged.

HXR Spike Search

find_peaks() scans HEL1OS for bursts above 4× noise floor.

HXR_MIN_PROM = 4.0

Event Matching
